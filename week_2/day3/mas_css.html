Unidades de medida

la unidad de medida mas usada son los pixeles(px) pero exites mas como los cm, mm, in(pulgadas), pt(texto en puntos), pc(picas). existen 2 grandes tipos de unidades que podemos usar las unidades absolutas y las unidades relativas que seran explicadas acontinuacion. por defecto se utiliza un tamaño de 16px algo importante es que cada una de las unidades de medida pueden ser trabajas con decimales por ejemplo 1.2px

unidades relativas

em: relativo al tamaño de fuente de un elemento si, por ejemplo, utilizamos 3em (font-size:3em); mi texto aumentara 3 veces su tamaño en relacion al tamaño que se este utilizando por defecto en la clase padre que lo contenga un ejemplo rapido seria si la clase padre tiene un font-zice de 32px 1em va valer 32px

rem:es igual a (em) con la diferenca de que este actua en la raiz o sea en la letra que este utilizando por defecto y para cambiarla abria que cambiarla con la etiqueta html en los ajustes generales del css
